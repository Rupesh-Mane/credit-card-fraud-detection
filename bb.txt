__init__.py (Very Simple)

Definition:

###########################################################
📄 Module
A single .py file.

Example:

config.py
ingestion.py
train.py

Memory: Module = File 📄
#####################################################
📁 Package
A folder containing Python modules.

Example:

src/
├── config.py
├── ingestion.py
└── train.py
 
Memory: Package = Folder 📁
##########################################################
👤 self
Refers to the current object of a class.

Example:

class DataIngestion:

    def load_data(self):
        pass

Here, self means this DataIngestion object.

Memory: self = Current Object

###############################################################

📂 __pycache__
A folder created by Python to make your code run faster.
Stores compiled .pyc files.

Memory: Python's Cache Folder

⚡ .pyc
What is .pyc?
.pyc = Compiled Python Code
Python converts your .py file into a .pyc file so it can run faster the next time.

Memory:
.py → Your code 📄
.pyc → Faster version ⚡

#####################################################################################
===========================
Running Python Files
===========================

1. Run a Python File 📄

Syntax:
python file.py

Example:
python app.py

Meaning:
Run a Python file.


---------------------------

2. Run a Python Module 📦

Syntax:
python -m package.module

Example:
python -m src.data.ingestion

Meaning:
Run a module inside a package.


---------------------------

❌ Wrong

python -m src.data.ingestion.py

Reason:
When using -m, never write .py.


---------------------------

Easy Memory Trick 🚀

python file.py
        ↓
Run a File 📄

python -m package.module
        ↓
Run a Module 📦


For My Project

python -m src.data.ingestion


One-Line Revision

Without -m → Run a File 📄

With -m → Run a Module 📦
